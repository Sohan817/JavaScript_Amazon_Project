# Javascript Amazon Project

This project is a JavaScript-based implementation inspired by core features of Amazon. It demonstrates fundamental concepts of web development, including product listing, cart functionality, and dynamic user interactions.

## Features

- Product listing and display
- Add to cart functionality
- Remove items from cart
- Dynamic price calculation
- Responsive UI

## Technologies Used

- HTML
- CSS
- JavaScript
